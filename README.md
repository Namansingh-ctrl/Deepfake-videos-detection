# Deepfake Videos Detection 🎭

## 📌 Overview

This project detects whether a video is **real or deepfake** using deep learning techniques.
It extracts faces from videos and uses a trained model to classify them.

---

## ⚙️ Features

* Face extraction from videos using OpenCV
* Deep learning model for classification
* Real-time prediction support
* Works on both images and videos

---

## 🧠 Tech Stack

* Python
* OpenCV
* NumPy
* PyTorch
* tqdm

## 🚀 How to Run

### 1. Clone the repository

```
git clone https://github.com/Namansingh-ctrl/Deepfake-videos-detection.git
cd Deepfake-videos-detection
```

### 2. Install dependencies

```
pip install -r requirements.txt
```

### 3. Run the project

```
python app.py
```
